# Веб-интерфейс курса по ИИ для начинающих

## 🚀 Быстрый старт

### Вариант 1: Открыть в браузере
Просто откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge, Safari).

### Вариант 2: Локальный сервер (рекомендуется)
```bash
# Если установлен Python 3
cd ai-course-web
python -m http.server 8000

# Затем откройте в браузере: http://localhost:8000
```

### Вариант 3: Live Server (для разработчиков)
Если используете VS Code:
1. Установите расширение "Live Server"
2. Откройте `index.html`
3. Нажмите "Go Live"

## 🛠️ Кастомизация

### Изменение цветовой схемы
Откройте `styles/main.css` и измените CSS-переменные:

```css
:root {
    --primary-color: #4f46e5;      /* Основной цвет */
    --secondary-color: #06b6d4;    /* Вторичный цвет */
    --accent-color: #f59e0b;       /* Акцентный цвет */
}
```

### Добавление нового модуля
1. Откройте `scripts/data.js`
2. Добавьте новый объект в `courseData.modules`
3. Обновите HTML в `index.html` (секция modules)

### Добавление термина в глоссарий
В файле `scripts/data.js` добавьте в массив `glossary`:

```javascript
{ term: "Новый термин", def: "Определение термина" }
```
