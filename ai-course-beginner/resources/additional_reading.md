# 📚 Дополнительные материалы и ресурсы

## 📖 Книги

### Для начинающих

1. **"Искусственный интеллект: современный подход"** — Стюарт Рассел, Питер Норвиг
   - Классический учебник по ИИ
   - Подходит для глубокого понимания основ
   - [Ссылка](https://www.ozon.ru/product/iskusstvennyy-intellekt-sovremennyy-podhod-1804763/)

2. **"Глубокое обучение. Погружение в мир нейронных сетей"** — Сет Вайдман
   - Доступное введение в глубокое обучение
   - Минимум математики, максимум практики

3. **"Python для сложных задач: наука о данных и машинное обучение"** — Джейк Вандер Плас
   - Практическое руководство по работе с данными на Python
   - Библиотеки: NumPy, Pandas, Matplotlib, Scikit-Learn

### Для продолжающих

4. **"Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow"** — Aurélien Géron
   - Практическое руководство с кодом
   - От классического ML до глубокого обучения
   - [GitHub с кодом](https://github.com/ageron/handson-ml2)

5. **"Deep Learning"** — Ian Goodfellow, Yoshua Bengio, Aaron Courville
   - "Библия" глубокого обучения
   - Более математически строгий подход
   - [Доступна онлайн бесплатно](https://www.deeplearningbook.org/)

6. **"Pattern Recognition and Machine Learning"** — Christopher Bishop
   - Глубокое понимание машинного обучения
   - Bayesian подход

### Популярная литература

7. **"Жизнь 3.0"** — Макс Тегмарк
   - О будущем ИИ и его влиянии на человечество
   - Философский взгляд

8. **"Суперинтеллект"** — Ник Бостром
   - Риски и возможности сверхразума
   - Этические вопросы

9. **"Homo Deus: Краткая история будущего"** — Юваль Ной Харари
   - Роль ИИ в будущем человечества

## 💼 Карьера в ИИ

### Роли в индустрии

1. **Data Analyst** — анализ данных, визуализация
2. **Data Scientist** — построение ML-моделей
3. **ML Engineer** — внедрение моделей в продакшен
4. **AI Researcher** — научные исследования
5. **Computer Vision Engineer** — работа с изображениями
6. **NLP Engineer** — обработка естественного языка

### Портфолио

Что включить в портфолио:
- 3-5 завершённых проекта
- Код на GitHub с README
- Участие в Kaggle соревнованиях
- Блог или статьи о проектах
- Сертификаты курсов

### Подготовка к собеседованиям

Ресурсы:
- **LeetCode** — задачи по алгоритмам
- **Kaggle** — практические навыки
- **Glassdoor** — вопросы с собеседований

## 🎯 План самообучения на 6 месяцев

### Месяц 1-2: Основы
- Python (если не знаете)
- Математика: линейная алгебра, статистика
- Библиотеки: NumPy, Pandas, Matplotlib
- Курс: "AI For Everyone"

### Месяц 3-4: Машинное обучение
- Scikit-Learn
- Классические алгоритмы ML
- Курс: Machine Learning (Andrew Ng)
- Первые проекты на Kaggle

### Месяц 5: Глубокое обучение
- Нейронные сети
- TensorFlow или PyTorch
- Курс: Deep Learning Specialization
- Проект: классификация изображений

### Месяц 6: Специализация
- Выберите направление (CV, NLP, etc.)
- Углублённое изучение
- Финальный проект для портфолио
- Подготовка резюме

## 🔍 Поиск информации

### Как гуглить эффективно

```
"конкретная ошибка" + название библиотеки
site:stackoverflow.com ваша проблема
"how to" + задача + python
filetype:pdf "deep learning" tutorial
```

### Вопросы на Stack Overflow

Перед вопросом:
1. Поищите, не задан ли уже такой вопрос
2. Приведите минимальный воспроизводимый пример
3. Покажите, что вы уже пробовали
4. Укажите версии библиотек
