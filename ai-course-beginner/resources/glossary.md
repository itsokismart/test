# 📖 Глоссарий терминов по ИИ

## A

**AI (Artificial Intelligence)** — Искусственный интеллект. Область компьютерных наук, создающая системы, способные выполнять задачи, требующие человеческого интеллекта.

**Algorithm** — Алгоритм. Пошаговая инструкция или набор правил для решения задачи.

**Activation Function** — Функция активации. Функция, определяющая выход нейрона на основе входных сигналов. Добавляет нелинейность в нейронную сеть. Примеры: ReLU, Sigmoid, Tanh.

**AGI (Artificial General Intelligence)** — Общий искусственный интеллект. Гипотетический ИИ, способный выполнять любые интеллектуальные задачи, доступные человеку.

**ANI (Artificial Narrow Intelligence)** — Узкий искусственный интеллект. ИИ, специализирующийся на одной конкретной задаче. Все современные системы ИИ относятся к этому типу.

**Attention Mechanism** — Механизм внимания. Техника, позволяющая модели фокусироваться на наиболее важных частях входных данных. Основа архитектуры Transformer.

**Autoencoder** — Автоэнкодер. Нейронная сеть, которая учится сжимать данные (энкодер) и восстанавливать их обратно (декодер). Используется для понижения размерности и генерации данных.

**Accuracy** — Точность. Метрика классификации: доля правильных предсказаний от общего числа. `Accuracy = (TP+TN)/(TP+TN+FP+FN)`

## B

**Backpropagation** — Обратное распространение ошибки. Алгоритм обучения нейронных сетей, вычисляющий градиенты функции потерь по всем весам сети.

**Bias** — Смещение. 
1. Параметр нейрона, позволяющий сдвигать функцию активации.
2. Систематическая ошибка модели (смещение-дисперсия).

**Batch** — Пакет. Группа примеров, обрабатываемых вместе перед обновлением весов модели.

**Batch Normalization** — Пакетная нормализация. Техника нормализации входов каждого слоя, ускоряющая обучение и повышающая стабильность.

**Binary Classification** — Бинарная классификация. Задача классификации с двумя классами (например, спам/не спам).

## C

**Classification** — Классификация. Задача предсказания категории/класса объекта.

**Clustering** — Кластеризация. Задача группировки похожих объектов без предварительных меток. Обучение без учителя.

**CNN (Convolutional Neural Network)** — Свёрточная нейронная сеть. Архитектура, эффективная для обработки изображений. Использует свёрточные слои для распознавания локальных паттернов.

**Cross-Entropy** — Перекрёстная энтропия. Функция потерь, часто используемая в задачах классификации.

**Cross-Validation** — Кросс-валидация. Метод оценки модели, при котором данные разбиваются на несколько частей, и модель обучается/тестируется на разных комбинациях.

## D

**Data Augmentation** — Аугментация данных. Техника увеличения обучающей выборки путём создания модифицированных версий существующих данных (поворот изображений, замена слов и т.д.).

**Deep Learning** — Глубокое обучение. Подраздел машинного обучения, использующий многослойные нейронные сети.

**Dropout** — Метод регуляризации, при котором случайные нейроны временно "отключаются" во время обучения для предотвращения переобучения.

**Decision Tree** — Дерево решений. Алгоритм, строящий дерево вопросов для принятия решений.

## E

**Epoch** — Эпоха. Один полный проход по всем обучающим данным.

**Ensemble Learning** — Ансамблевое обучение. Комбинирование нескольких моделей для улучшения предсказаний. Примеры: Random Forest, Gradient Boosting.

**Exploding Gradient** — Взрывающийся градиент. Проблема, когда градиенты становятся экспоненциально большими, делая обучение нестабильным.

## F

**Feature** — Признак. Характеристика или свойство объекта, используемое моделью для предсказания.

**Feature Engineering** — Создание признаков. Процесс создания новых признаков из существующих данных для улучшения качества модели.

**F1-Score** — Гармоническое среднее между Precision и Recall. `F1 = 2*(Precision*Recall)/(Precision+Recall)`

**Forward Propagation** — Прямое распространение. Проход данных через сеть от входа к выходу.

**Fully Connected Layer** — Полносвязный слой. Слой нейронной сети, где каждый нейрон связан со всеми нейронами предыдущего слоя.

## G

**GAN (Generative Adversarial Network)** — Генеративно-состязательная сеть. Архитектура из двух сетей (генератор и дискриминатор), соревнующихся друг с другом.

**Gradient Descent** — Градиентный спуск. Алгоритм оптимизации, минимизирующий функцию потерь путём движения в направлении отрицательного градиента.

**GPU (Graphics Processing Unit)** — Графический процессор. Специализированный процессор, эффективно выполняющий параллельные вычисления. Широко используется для обучения нейронных сетей.

## H

**Hidden Layer** — Скрытый слой. Слои нейронной сети между входным и выходным слоями.

**Hyperparameter** — Гиперпараметр. Параметр, задаваемый до обучения модели (скорость обучения, количество слоёв и т.д.), в отличие от весов, которые обучаются.

## K

**KNN (K-Nearest Neighbors)** — Метод K-ближайших соседей. Алгоритм, классифицирующий объект по большинству классов среди его K ближайших соседей.

**K-Means** — Алгоритм кластеризации, разделяющий данные на K кластеров.

## L

**Label** — Метка. Правильный ответ для примера в обучающих данных.

**Learning Rate** — Скорость обучения. Гиперпараметр, определяющий размер шага при обновлении весов.

**Linear Regression** — Линейная регрессия. Алгоритм регрессии, предполагающий линейную зависимость между признаками и целевой переменной.

**Logistic Regression** — Логистическая регрессия. Алгоритм бинарной классификации, использующий сигмоидную функцию.

**Loss Function** — Функция потерь. Функция, измеряющая ошибку модели (разницу между предсказанием и правильным ответом).

**LSTM (Long Short-Term Memory)** — Тип рекуррентной нейронной сети, способный запоминать долгосрочные зависимости.

## M

**Machine Learning** — Машинное обучение. Подраздел ИИ, изучающий методы построения алгоритмов, обучающихся на данных.

**MSE (Mean Squared Error)** — Среднеквадратичная ошибка. Метрика регрессии: среднее квадратов разностей между предсказаниями и истинными значениями.

**Multi-class Classification** — Многоклассовая классификация. Задача классификации с тремя и более классами.

## N

**Neural Network** — Нейронная сеть. Вычислительная система, состоящая из взаимосвязанных узлов (нейронов), вдохновлённая биологическими нейронными сетями.

**Neuron** — Нейрон. Базовая единица нейронной сети, принимающая входы, применяющая веса и функцию активации, выдающая выход.

**NLP (Natural Language Processing)** — Обработка естественного языка. Область ИИ, занимающаяся взаимодействием компьютеров и человеческого языка.

**Normalization** — Нормализация. Приведение данных к единому масштабу (например, от 0 до 1).

## O

**Overfitting** — Переобучение. Ситуация, когда модель слишком хорошо запоминает обучающие данные (включая шум) и плохо обобщает на новые данные.

**Optimizer** — Оптимизатор. Алгоритм обновления весов модели для минимизации функции потерь. Примеры: SGD, Adam, RMSprop.

## P

**Perceptron** — Перцептрон. Простейшая нейронная сеть из одного нейрона.

**Precision** — Точность. Метрика классификации: доля истинно положительных среди всех предсказанных положительных. `Precision = TP/(TP+FP)`

**Preprocessing** — Предобработка. Подготовка данных перед обучением модели (очистка, нормализация, кодирование).

## R

**R² (R-squared)** — Коэффициент детерминации. Метрика регрессии, показывающая долю дисперсии, объяснённую моделью.

**Recall** — Полнота. Метрика классификации: доля истинно положительных среди всех фактически положительных. `Recall = TP/(TP+FN)`

**Regression** — Регрессия. Задача предсказания непрерывного числового значения.

**Regularization** — Регуляризация. Техники предотвращения переобучения путём добавления штрафа за сложность модели (L1, L2).

**Reinforcement Learning** — Обучение с подкреплением. Тип обучения, где агент обучается через взаимодействие со средой, получая награды или штрафы.

**RNN (Recurrent Neural Network)** — Рекуррентная нейронная сеть. Архитектура для обработки последовательностей, имеющая циклические связи.

**ROC-AUC** — Площадь под ROC-кривой. Метрика классификации, оценивающая способность модели различать классы.

**Random Forest** — Случайный лес. Ансамблевый алгоритм, комбинирующий множество деревьев решений.

## S

**SGD (Stochastic Gradient Descent)** — Стохастический градиентный спуск. Вариант градиентного спуска, обновляющий веса после каждого примера.

**Sigmoid** — Сигмоидная функция активации: `f(x) = 1/(1+e^(-x))`. Выдаёт значения от 0 до 1.

**Softmax** — Функция активации для многоклассовой классификации. Преобразует выходы в вероятности, сумма которых равна 1.

**Supervised Learning** — Обучение с учителем. Обучение на размеченных данных с известными правильными ответами.

**SVM (Support Vector Machine)** — Метод опорных векторов. Алгоритм классификации, находящий оптимальную границу между классами.

## T

**Test Set** — Тестовая выборка. Часть данных, используемая для финальной оценки качества модели.

**Transfer Learning** — Трансферное обучение. Использование предобученной модели для новой, но схожей задачи.

**Transformer** — Архитектура нейронной сети, основанная на механизме внимания. Лежит в основе современных языковых моделей (GPT, BERT).

**Training Set** — Обучающая выборка. Часть данных, используемая для обучения модели.

**Turing Test** — Тест Тьюринга. Метод определения способности машины мыслить, предложенный Аланом Тьюрингом.

## U

**Underfitting** — Недообучение. Ситуация, когда модель слишком проста и не улавливает закономерности в данных.

**Unsupervised Learning** — Обучение без учителя. Обучение на неразмеченных данных, где алгоритм сам ищет структуры.

## V

**Validation Set** — Валидационная выборка. Часть данных, используемая для настройки гиперпараметров и выбора лучшей модели.

**Vanishing Gradient** — Затухающий градиент. Проблема, когда градиенты становятся экспоненциально малыми, замедляя или останавливая обучение.

## W

**Weight** — Вес. Параметр нейрона, определяющий важность соответствующего входа.
