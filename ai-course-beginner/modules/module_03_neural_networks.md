## 1. Что такое Нейронная Сеть?

**Нейронная сеть (Neural Network)** — это вычислительная система, вдохновлённая биологическими нейронными сетями мозга, состоящая из взаимосвязанных узлов (нейронов), которые обрабатывают информацию.

### Биологический нейрон vs Искусственный нейрон

#### Биологический нейрон:
```
Дендриты → Тело клетки → Аксон → Синапсы → Другие нейроны
     (вход)    (обработка)    (выход)
```

#### Искусственный нейрон:
```
Входы (x₁, x₂, ..., xₙ) 
       ↓
Веса (w₁, w₂, ..., wₙ)
       ↓
Сумматор: Σ(wᵢ * xᵢ) + b
       ↓
Функция активации: f(Σ)
       ↓
Выход: y
```

### Формула искусственного нейрона:

```
y = f(w₁x₁ + w₂x₂ + ... + wₙxₙ + b)
```

Где:
- `xᵢ` — входные значения
- `wᵢ` — веса (важность каждого входа)
- `b` — смещение (bias)
- `f` — функция активации
- `y` — выходное значение

## 2. Архитектура Нейронной Сети

### Основные компоненты:

#### 1. Входной слой (Input Layer)
- Получает исходные данные
- Количество нейронов = количество признаков
- Не выполняет вычислений, только передаёт данные

#### 2. Скрытые слои (Hidden Layers)
- Выполняют основные вычисления
- Могут быть один или много слоёв
- Каждый нейрон связан с нейронами предыдущего и следующего слоя

#### 3. Выходной слой (Output Layer)
- Выдаёт окончательный результат
- Количество нейронов зависит от задачи:
  - 1 нейрон для регрессии
  - 2+ нейронов для классификации (по одному на класс)

### Пример архитектуры:

```
Входной слой    Скрытый слой 1   Скрытый слой 2   Выходной слой
     ○                ○○○              ○○○○            ○
     ○                ○○○              ○○○○            ○
     ○      →         ○○○     →        ○○○○     →      ○
     ○                ○○○              ○○○○
     ○                ○○○

   5 нейронов       4 нейрона        4 нейрона       3 нейрона
  (признаки)                          (скрытые)      (классы)
```

### Типы архитектур:

#### Полносвязная сеть (Fully Connected / Dense)
- Каждый нейрон связан со всеми нейронами следующего слоя
- Подходит для табличных данных

#### Сверточная сеть (CNN - Convolutional Neural Network)
- Использует свёрточные слои
- Специализирована для изображений
- Распознаёт локальные паттерны

#### Рекуррентная сеть (RNN - Recurrent Neural Network)
- Имеет связи назад во времени
- Обрабатывает последовательности
- Подходит для текста, временных рядов

#### Трансформер (Transformer)
- Использует механизм внимания (attention)
- Параллельная обработка последовательностей
- Основа современных языковых моделей (GPT, BERT)

## 3. Функции Активации

**Функция активации** определяет, должен ли нейрон "активироваться" и передать сигнал дальше. Добавляет нелинейность в сеть.

### Популярные функции активации:

#### 1. Sigmoid (Сигмоида)
```
f(x) = 1 / (1 + e^(-x))
```
- **Диапазон:** от 0 до 1
- **Применение:** бинарная классификация (выходной слой)
- **Преимущества:** интерпретируется как вероятность
- **Недостатки:** проблема затухающих градиентов

```
График:
  1 │        ╱‾‾‾‾
    │      ╱
0.5│    ╱
    │  ╱
  0 │╱
    └─────────→ x
```

#### 2. Tanh (Гиперболический тангенс)
```
f(x) = (e^x - e^(-x)) / (e^x + e^(-x))
```
- **Диапазон:** от -1 до 1
- **Применение:** скрытые слои
- **Преимущества:** центрирована вокруг 0
- **Недостатки:** также затухающие градиенты

#### 3. ReLU (Rectified Linear Unit)
```
f(x) = max(0, x)
```
- **Диапазон:** от 0 до ∞
- **Применение:** скрытые слои (самая популярная)
- **Преимущества:** 
  - Быстро вычисляется
  - Решает проблему затухающих градиентов
- **Недостатки:** "мёртвые нейроны" (выход всегда 0)

```
График:
    │  ╱
    │ ╱
    │╱
────┼─────────→ x
    │
```

#### 4. Leaky ReLU
```
f(x) = x, если x > 0
f(x) = 0.01x, если x ≤ 0
```
- **Преимущества:** решает проблему мёртвых нейронов
- **Применение:** альтернатива ReLU

#### 5. Softmax
```
f(xᵢ) = e^(xᵢ) / Σe^(xⱼ)
```
- **Диапазон:** от 0 до 1, сумма всех выходов = 1
- **Применение:** многоклассовая классификация (выходной слой)
- **Интерпретация:** вероятность каждого класса

### Сравнение функций активации:

| Функция | Диапазон | Применение | Преимущества | Недостатки |
|---------|----------|------------|--------------|------------|
| Sigmoid | 0...1 | Выход (бинарный) | Вероятность | Затухание градиента |
| Tanh | -1...1 | Скрытые слои | Центрирована | Затухание градиента |
| ReLU | 0...∞ | Скрытые слои | Быстрая, эффективная | Мёртвые нейроны |
| Leaky ReLU | -∞...∞ | Скрытые слои | Нет мёртвых нейронов | Нужно подбирать α |
| Softmax | 0...1 (сумма=1) | Выход (мультикласс) | Вероятности | Только для выхода |

## 4. Процесс Обучения Нейронной Сети

### Этапы обучения:

#### 1. Инициализация весов
- Веса случайным образом инициализируются небольшими значениями
- Важность правильной инициализации для сходимости

#### 2. Прямое распространение (Forward Propagation)
```
Вход → Слой 1 → Слой 2 → ... → Выход
```
- Данные проходят через сеть от входа к выходу
- На каждом слое применяется: `y = f(Wx + b)`

#### 3. Вычисление функции потерь (Loss Function)
- Сравниваем предсказание с правильным ответом
- Измеряем ошибку модели

**Популярные функции потерь:**

**Для регрессии:**
- MSE (Mean Squared Error): `L = (1/n) * Σ(y_pred - y_true)²`
- MAE (Mean Absolute Error): `L = (1/n) * Σ|y_pred - y_true|`

**Для классификации:**
- Binary Cross-Entropy: `L = -[y*log(p) + (1-y)*log(1-p)]`
- Categorical Cross-Entropy: `L = -Σ yᵢ * log(pᵢ)`

#### 4. Обратное распространение ошибки (Backpropagation)
- **Ключевой алгоритм обучения!**
- Вычисляем градиенты функции потерь по всем весам
- Используем цепное правило дифференцирования

**Как работает:**
```
1. Вычисляем ошибку на выходе
2. Распространяем ошибку назад через сеть
3. Для каждого веса вычисляем: ∂Loss/∂w
4. Обновляем веса в направлении, уменьшающем ошибку
```

**Математически:**
```
∂Loss/∂w = ∂Loss/∂output * ∂output/∂input * ∂input/∂w
```

#### 5. Обновление весов (Градиентный спуск)
```
w_new = w_old - learning_rate * ∂Loss/∂w
```

Где:
- `learning_rate` — скорость обучения (как сильно меняем веса)
- `∂Loss/∂w` — градиент (направление и величина изменения)

### Варианты градиентного спуска:

#### 1. Стохастический градиентный спуск (SGD)
- Обновляет веса после каждого примера
- Быстрый, но шумный

#### 2. Мини-батч градиентный спуск
- Обновляет веса после небольшой группы примеров (batch)
- Баланс между скоростью и стабильностью
- **Наиболее популярный вариант**

#### 3. Пакетный градиентный спуск
- Обновляет веса после всех примеров
- Медленный, требует много памяти

### Оптимизаторы (улучшенные версии градиентного спуска):

| Оптимизатор | Описание | Преимущества |
|-------------|----------|--------------|
| SGD | Базовый градиентный спуск | Простой |
| Momentum | Добавляет инерцию | Быстрее сходится |
| Adam | Адаптивная скорость обучения | Работает хорошо по умолчанию |
| RMSprop | Адаптирует learning rate | Хорош для RNN |
| AdaGrad | Накопление градиентов | Хорош для разреженных данных |

## 5. Глубокое Обучение

**Глубокое обучение (Deep Learning)** — подраздел машинного обучения, использующий многослойные (глубокие) нейронные сети.

### Чем глубокое обучение отличается от обычного МО?

| Обычное МО | Глубокое обучение |
|------------|-------------------|
| Требует ручного извлечения признаков | Автоматически извлекает признаки |
| Хорошо работает на небольших данных | Требует много данных |
| Быстрое обучение | Долгое обучение, нужны GPU |
| Интерпретируемые модели | "Чёрный ящик" |

### Почему "глубокое"?
- Много скрытых слоёв (обычно больше 2-3)
- Иерархическое представление признаков:
  - Первые слои: простые паттерны (края, углы)
  - Средние слои: комбинации паттернов (формы)
  - Последние слои: сложные концепции (объекты)

### Пример распознавания лица:

```
Слой 1: Края и линии
        ↓
Слой 2: Простые формы (круги, овалы)
        ↓
Слой 3: Части лица (глаза, нос, рот)
        ↓
Слой 4: Целое лицо
```

## 6. Типы Нейронных Сетей

### 1. Полносвязные сети (Feedforward Neural Networks)
- Базовая архитектура
- Информация движется только вперёд
- Применение: классификация, регрессия

### 2. Сверточные нейронные сети (CNN)
- Используют свёрточные слои
- Эффективны для изображений
- Архитектуры: LeNet, AlexNet, VGG, ResNet

**Применение:**
- Распознавание изображений
- Детекция объектов
- Сегментация изображений
- Медицинская диагностика

### 3. Рекуррентные нейронные сети (RNN)
- Имеют циклические связи
- Запоминают предыдущие состояния
- Варианты: LSTM, GRU

**Применение:**
- Обработка текста
- Распознавание речи
- Прогнозирование временных рядов
- Машинный перевод

### 4. Трансформеры (Transformers)
- Механизм внимания (Attention)
- Параллельная обработка
- Архитектуры: GPT, BERT, T5

**Применение:**
- Языковые модели
- Перевод
- Генерация текста
- Вопрос-ответ системы

### 5. Генеративно-состязательные сети (GAN)
- Две сети: генератор и дискриминатор
- Соревнуются друг с другом

**Применение:**
- Генерация изображений
- Стилизация
- Увеличение разрешения
- Создание глубоких фейков

### 6. Автоэнкодеры (Autoencoders)
- Сжимают и восстанавливают данные
- Архитектура: энкодер → бутылочное горлышко → декодер

**Применение:**
- Понижение размерности
- Шумоподавление
- Генерация данных
- Обнаружение аномалий

## 7. Практические Советы

### Как улучшить обучение нейронной сети:

#### 1. Подготовка данных
- Нормализуйте/стандартизируйте данные
- Используйте аугментацию данных (для изображений)
- Балансируйте классы

#### 2. Архитектура
- Начинайте с простой модели
- Постепенно усложняйте
- Используйте Batch Normalization

#### 3. Регуляризация (борьба с переобучением)
- **Dropout**: случайное отключение нейронов
- **L1/L2 регуляризация**: штраф за большие веса
- **Early Stopping**: остановка при ухудшении на валидации
- **Data Augmentation**: увеличение данных

#### 4. Гиперпараметры
- **Learning Rate**: начните с 0.001 (Adam)
- **Batch Size**: 32, 64, 128
- **Количество эпох**: используйте Early Stopping
- **Количество слоёв и нейронов**: экспериментируйте

#### 5. Отладка
- Следите за графиками loss и accuracy
- Используйте валидационную выборку
- Визуализируйте предсказания
- Проверяйте на переобучение
